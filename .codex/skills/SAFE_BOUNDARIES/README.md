# SAFE_BOUNDARIES (항상 적용)

이 프로젝트(AION2 HUB)는 **게임 클라이언트에 대한 접근/변조/자동화**를 절대 구현하지 않습니다.

## 금지(Non-goal)

- 게임 클라이언트/프로세스/메모리/패킷/후킹/변조/디컴파일/리버싱 관련 기능
- 자동 입력, 오토사냥, 외부 매크로(키보드/마우스 자동화 포함)
- 계정/보호조치/차단/정책 우회, VPN/하드웨어 차단 회피 등
- “정상 이용자가 할 수 없는 행위”를 가능하게 하는 기능 전반

## 허용(Goal)

- **로컬-퍼스트 플래너**(숙제 체크/기록/통계): 유저가 직접 입력/체크
- **공식 공지/업데이트 diff 요약**: 공개 웹 페이지 스냅샷/비교/알림
- **레기온 운영 도구**: 일정/모집/출석/리마인드(Discord 연동)

## 리뷰 규칙

- 위 금지 영역과 닿는 코드/아이디어는 “기술적으로 가능”해도 즉시 거부합니다.
